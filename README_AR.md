# منصة المزادات الوطنية - Algeria National Auction Platform

تطبيق مزادات احترافي معتمد من الدولة الجزائرية، مبني باستخدام Flutter

## 🎯 نظرة عامة

منصة المزادات الوطنية هي تطبيق متكامل لإدارة المزادات في الجزائر، مصمم خصيصاً للتجار والشركات المعتمدة فقط. يوفر النظام بيئة آمنة وموثوقة للمزادات بإشراف حكومي.

## 👥 الأدوار الرئيسية

### 1. البائع (Auctioneer) 🏪
**المسؤوليات:**
- إنشاء مزادات جديدة
- عرض السلع والمنتجات للمزاد
- متابعة مزاداته الخاصة
- الحصول على تقارير المبيعات

**المتطلبات:**
- سجل تجاري ساري المفعول
- رقم تعريف ضريبي (NIF)
- عنوان تجاري موثق
- موافقة الإدارة

### 2. المزايد (Bidder) 💼
**المسؤوليات:**
- المشاركة في المزادات النشطة
- تقديم مزايدات على السلع
- متابعة المزايدات الحالية
- شراء العناصر الفائزة

**المتطلبات:**
- تاجر فردي أو شركة مسجلة
- سجل تجاري
- رقم تعريف ضريبي (NIF)
- موافقة الإدارة

**أنواع الشركات المدعومة:**
- تاجر فردي (Individual Trader)
- شركة ذات مسؤولية محدودة (SARL)
- شركة مساهمة (SPA)
- أنواع أخرى

### 3. مدير المزاد (Auction Manager) 👨‍💼
**المسؤوليات:**
- مراجعة طلبات التسجيل الجديدة
- الموافقة على أو رفض المستخدمين
- مراجعة المزادات قبل نشرها
- الموافقة على أو رفض المزادات
- مراقبة النظام
- حل النزاعات

**الصلاحيات:**
- الوصول الكامل لجميع المزادات
- عرض بيانات جميع المستخدمين
- إدارة النظام بالكامل

## 📋 المميزات الرئيسية

## 🚀 دليل الاستخدام

### للبائعين:

#### 1. التسجيل
```
1. افتح التطبيق
2. اضغط على "إنشاء حساب"
3. اختر "بائع"
4. أدخل معلومات الشركة:
   - الاسم الكامل
   - البريد الإلكتروني
   - رقم الهاتف
   - اسم الشركة
   - نوع الشركة
   - رقم السجل التجاري
   - رقم التعريف الضريبي (NIF)
   - العنوان التجاري
   - المدينة والولاية
5. انتظر موافقة الإدارة (24-48 ساعة)
```

#### 2. إنشاء مزاد
```
1. بعد الموافقة على حسابك
2. اضغط على أيقونة "+" في الشاشة الرئيسية
3. املأ تفاصيل المزاد:
   - العنوان
   - الوصف التفصيلي
   - الفئة
   - السعر الابتدائي
   - السعر الاحتياطي (اختياري)
   - الموقع
   - الحالة
   - تاريخ ووقت الانتهاء
4. إرسال للمراجعة
5. انتظر موافقة المدير
```

### للمزايدين:

#### 1. التسجيل
```
1. افتح التطبيق
2. اضغط على "إنشاء حساب"
3. اختر "مزايد"
4. أدخل معلومات الشركة/التاجر
5. انتظر موافقة الإدارة
```

#### 2. المزايدة
```
1. تصفح المزادات النشطة
2. اختر المزاد المطلوب
3. اضغط على "المزايدة"
4. أدخل مبلغاً أعلى من السعر الحالي
5. قدم المزايدة
```

### للمديرين:

#### 1. مراجعة المستخدمين الجدد
```
1. افتح لوحة التحكم
2. اذهب إلى "طلبات التحقق من المستخدمين"
3. راجع معلومات كل طلب:
   - بيانات المستخدم
   - معلومات الشركة
   - السجل التجاري
   - الرقم الضريبي
4. وافق أو ارفض مع ذكر السبب
```

#### 2. مراجعة المزادات
```
1. افتح لوحة التحكم
2. اذهب إلى "طلبات الموافقة على المزادات"
3. راجع تفاصيل كل مزاد
4. تأكد من:
   - مطابقة المعلومات
   - صحة الأسعار
   - وضوح الوصف
5. وافق أو ارفض مع ملاحظات
```
