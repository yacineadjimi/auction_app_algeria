# National Auction - تطبيق إدارة المزادات

تطبيق احترافي لإدارة المزادات مبني باستخدام Flutter

## المميزات

### ✨ الشاشات الرئيسية
1. **شاشة تسجيل الدخول (Login)**
   - تسجيل الدخول بالبريد الإلكتروني وكلمة المرور
   - خيار "تذكرني"
   - استرجاع كلمة المرور
   - تصميم عصري وجذاب

2. **شاشة إنشاء حساب (Sign Up)**
   - تسجيل حساب جديد
   - حقول: الاسم الكامل، البريد الإلكتروني، رقم الهاتف، كلمة المرور
   - الموافقة على الشروط والأحكام
   - التحقق من صحة البيانات

3. **الشاشة الرئيسية (Home)**
   - عرض المزادات النشطة
   - إحصائيات شاملة:
     - عدد المزادات النشطة
     - إجمالي المزايدات
     - القيمة الإجمالية
   - بحث عن المزادات
   - تصفية حسب الفئات
   - عرض تفاصيل كل مزاد:
     - الصورة
     - العنوان والوصف
     - السعر الحالي
     - الوقت المتبقي
     - عدد المزايدات
   - إمكانية المزايدة على أي مزاد

## تثبيت المشروع

### المتطلبات
- Flutter SDK (الإصدار 3.0.0 أو أحدث)
- Dart SDK
- محرر نصوص (VS Code أو Android Studio)

### خطوات التثبيت

1. **استنساخ المشروع**
```bash
# إذا كان لديك git repository
git clone <your-repo-url>
cd auction_app
```

2. **تثبيت الحزم**
```bash
flutter pub get
```

3. **تشغيل التطبيق**
```bash
# للأندرويد
flutter run

# للـ iOS
flutter run -d ios

# للويب
flutter run -d chrome
```

## بنية المشروع

```
lib/
├── main.dart                 # نقطة بداية التطبيق
├── models/                   # نماذج البيانات
│   ├── auction.dart         # نموذج المزاد
│   └── user.dart            # نموذج المستخدم
├── providers/               # إدارة الحالة
│   └── auction_provider.dart
└── screens/                 # الشاشات
    ├── login_screen.dart    # شاشة تسجيل الدخول
    ├── signup_screen.dart   # شاشة التسجيل
    └── home_screen.dart     # الشاشة الرئيسية
```

## الحزم المستخدمة

- **provider**: إدارة الحالة
- **intl**: تنسيق التواريخ والأرقام

## التخصيص

### تغيير الألوان
افتح ملف `lib/main.dart` وعدل في `ThemeData`:

```dart
theme: ThemeData(
  primarySwatch: Colors.teal, // غير اللون الأساسي هنا
  // ...
),
```

### إضافة مزادات جديدة
افتح `lib/providers/auction_provider.dart` وعدل في دالة `_loadDummyData()`.
